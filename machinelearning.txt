1. Machine Learning?
Machine Learning is the ability to automatically learn and improve from experience without being explicitly programmed. Machine Learning focuses on the development of computer programs that can access data and use it learn for themselves.

---> Subset of Artificial Intelligence(AI) which provides machines the ability to learn automatically & improve from experience without being explicitly programmed.

--->Traditional Programming:
Rules + Data ---> |Traditional Programming| ---> Answer

---> Machine Learning:
Answer + Data ----> |Machine Learning| ---> Rules
